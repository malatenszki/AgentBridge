## Description

### Version 1 (Concise - Recommended)
```
Agent Bridge connects your iPhone to AI agents running on your Mac, letting you monitor terminal sessions and respond to prompts from anywhere in your home.

FEATURES

• Real-Time Monitoring
Stream terminal output directly to your iPhone as it happens

• Smart Notifications
Get alerts when agents need input or complete tasks

• Quick Responses
Respond to prompts with one tap or send custom commands

• Session Management
View and manage multiple agent sessions simultaneously

• Private & Secure
All communication stays on your local WiFi network
No data collection, no cloud servers

• Simple Setup
Scan a QR code to pair your devices and start monitoring in seconds

HOW IT WORKS

1. Install the daemon on your Mac
2. Scan the QR code with your iPhone
3. Monitor AI agent sessions from anywhere

Perfect for keeping track of long-running Claude Code sessions, development tasks, and automation scripts without being tied to your desk.

REQUIREMENTS

• macOS 13.0 or later on your Mac
• Same WiFi network for both devices
• Free daemon app for Mac (installation instructions in the app)

PRIVACY

Agent Bridge respects your privacy. All data stays on your local network. We don't collect, store, or transmit any personal information. Your terminal sessions are yours alone.
```

## App Review Notes (For Apple Reviewers)

```
TESTING INSTRUCTIONS

To test Agent Bridge, you'll need both the iOS app and the macOS daemon:

1. INSTALL MACOS DAEMON
   The macOS daemon is required to test the app.

   Build from source:
   - Clone: https://github.com/malatenszki/AgentBridge
   - Navigate to macos-daemon/
   - Run: swift build -c release
   - Run: .build/release/agent-bridge

2. PAIR DEVICES
   - Open Agent Bridge on iOS
   - Tap "Pair New Device" in Settings
   - Scan the QR code displayed in the Terminal
   - Connection should establish within 5 seconds

3. CREATE A TEST SESSION
   - In Terminal on Mac, run any command (e.g., "ls -la")
   - The session should appear in the iOS app
   - You should see real-time output

4. TEST NOTIFICATIONS
   - Enable notifications in Settings
   - Run a command that completes quickly
   - You should receive a notification

DEMO CREDENTIALS
Not applicable - pairing is done via QR code, no credentials needed.

PRIVACY
This app does not collect any user data. All communication is local network only.
No analytics or tracking is used.

NOTES
• Requires both devices on same WiFi network
• macOS daemon must be running for functionality
• All features work offline (no internet required)
• App is fully functional without any external services

Questions? Contact: review@agentbridge.io
```
