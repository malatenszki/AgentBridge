Agent Bridge Support

Need help? You're in the right place. This guide covers setup, troubleshooting, and common questions.

Quick Start

1. Get the macOS Daemon

Download from GitHub:

git clone https://github.com/malatenszki/agent-bridge.git
cd AgentBridge/macos-daemon
swift build -c release

2. Start the Daemon

.build/release/agent-bridge

3. Pair Your iPhone

  • Open the Agent Bridge app on your iPhone
  • Tap "Pair New Device" in Settings
  • Scan the QR code shown in the Terminal

System Requirements

  • macOS: macOS 13.0 (Ventura) or later
  • iOS: iOS 16.0 or later
  • Network: Both devices must be on the same WiFi network
  • Port: Port 8765 must be available (configurable)

Frequently Asked Questions

Why can't I see the pairing QR code?

Make sure:

  • The daemon is running (agent-bridge)
  • You're running it in a terminal with QR code support
  • Port 8765 is not already in use

The QR code expired - what do I do?

Pairing codes expire after 60 seconds for security. Simply restart the daemon to generate a new QR code:

agent-bridge

I can't connect to my Mac

Check these common issues:

  • WiFi: Both devices must be on the same WiFi network (not cellular data)
  • Firewall: macOS Firewall might be blocking connections - allow agent-bridge in System Settings → Network → Firewall
  • Daemon Running: Make sure agent-bridge is still running on your Mac
  • Port: Another app might be using port 8765 - try restarting the daemon

How do I enable notifications?

  1. Open Agent Bridge on iPhone
  2. Go to Settings (gear icon)
  3. Toggle "Push Notifications"
  4. Allow notifications when iOS prompts you

Can I use this over the internet?

No. Agent Bridge is designed to work only on your local WiFi network for security and privacy. Your data never leaves your local network.

How do I unpair my iPhone?

  1. Open Agent Bridge on iPhone
  2. Go to Settings
  3. Tap "Unpair"
  4. Confirm the action

Troubleshooting

Daemon Won't Start

Error: Port already in use

Another process is using port 8765. Find and stop it:

lsof -i :8765
kill [PID from above command]

Or specify a different port:

AGENT_BRIDGE_PORT=8766 agent-bridge

Connection Drops Frequently

  • Check WiFi signal strength on both devices
  • Ensure your Mac doesn't go to sleep (disable sleep in System Settings)
  • Check for WiFi interference or network issues

Sessions Not Appearing

  • Refresh by pulling down on the session list
  • Check that the daemon is still running
  • Restart the iPhone app
⚠️ Firewall Warning

If you're using macOS Firewall or third-party security software, you may need to allow incoming connections for agent-bridge.

Advanced Configuration

Custom Port

export AGENT_BRIDGE_PORT=8766
agent-bridge

Daemon Logs

The daemon outputs logs to the terminal. For verbose logging, check the daemon's standard output.

Known Limitations

  • Requires WiFi connection (cellular data not supported)
  • Works only on local network (no remote access)
  • macOS daemon must remain running for connections

Privacy & Security

Agent Bridge takes your privacy seriously:

  • All data stays on your local network
  • No cloud services or external servers
  • Pairing tokens expire after 60 seconds
  • Secure storage using Keychain
